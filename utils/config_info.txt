"name" - В это поле нужно ввести название для вашего ассистента по которому вы будете к нему обращаться
"chatgpt" - В это поле можно ввести любую версию chatgpt рекомендуется gpt-4o
"openai_apikey" - В это поле нужно ввести api ключ от вашего аккаунта openai P.S. чтобы работал gpt-4o нужен оплаченый ключ хотя бы на $ 5
"recogniz" - Сюда можно ввести значение распознования speach_recognize / vosk

"microphone" 
    "active" - Ваш индекс микрофона с которого вы будете говорить P.S. чтобы получить индекс микрофона включите в настройках отображение индексов микрофона и запустите асситента

"telegram"
    "API_TOKEN" - Api токен телеграм от вашего бота. Получить бота можно тут - @BotFather
    "ALLOWED_USER" - Ваш телеграм id чтобы только вы могли использовать бота. Получить id можно тут - @my_id_bot

"SSH"
    "ip" - ip сервера
    "username" - username
    "password" - пароль
    "path_send" - путь папки на сервере

    в эти строки загружаются данные от сервера для протокола 21 по бекапу нужных файлов с компьютера
